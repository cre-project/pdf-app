# pdf-app

1) set env variables 
```
    FIREBASE_API_KEY,
    FIREBASE_DOMAIN,
    FIREBASE_DB_URL,
    FIREBASE_PROJECT_ID,
    FIREBASE_DB_URL
```
2) run `DEBUG=app:* npm start`
3) application will run on port 3000
4) go to `http://localhost:3000/{"valuationId": "valuationId", "userId": "userId"}`
